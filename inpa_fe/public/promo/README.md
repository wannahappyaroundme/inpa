# 판촉물 디자인 시안 넣는 곳 (PM 가이드)

이 폴더(`inpa_fe/public/promo/`)에 디자인 시안 이미지를 **아래 이름 규칙대로** 넣으면,
시드(`seed_promotion`)가 판촉물 **미리보기 화면·샘플 사진**으로 자동 연결합니다.
(Next.js public 폴더라 `/promo/<파일명>` 주소로 바로 서빙됩니다.)

## 파일 형식
- 권장: `.webp` 또는 `.png`(투명배경 가능) / `.jpg`도 OK
- 잘리지 않게 **긴 쪽 100%**로 표시되니(object-contain), 시안 자체 비율 그대로 두면 됩니다.
  세로형이면 세로가 길게, 가로형이면 가로가 길게.

## 이름 규칙 (영문, 소문자, 하이픈)
`<유형>-<세부>-<번호>.<확장자>` · 번호는 같은 종류 여러 시안일 때 1,2,3…

| 판촉물 | 세부 | 파일명 예시 |
|---|---|---|
| 명함 | 가로형 | `card-h-1.png`, `card-h-2.png` |
| 명함 | 세로형 | `card-v-1.png` |
| 달력 | 탁상용 | `calendar-desk-1.png` |
| 달력 | 벽걸이용 | `calendar-wall-1.png` |
| 리플렛 | — | `leaflet-1.png` |
| 팜플렛 | — | `pamphlet-1.png` |
| 파일보관함 | — | `file-holder-1.png` |

> 한 샘플에 여러 장(앞/뒤 등)을 넣고 싶으면 같은 prefix에 번호만 늘리세요
> (`card-h-1.png`, `card-h-1-back.png` 식). 첫 번째(`-1`)가 미리보기 대표 이미지가 됩니다.

## 넣은 뒤
파일을 이 폴더에 두고 알려주시면, 제가 `seed_promotion`에 그 파일을 매핑해
샘플(명칭·카테고리·세부·재질/사이즈 설명)을 만들고, 배포 시 자동 적재되게 합니다.
미리보기·샘플 사진이 바로 그 시안으로 바뀝니다.
