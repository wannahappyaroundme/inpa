# Spec: new.inpa.kr 시네마틱 브랜드 랜딩 (하이브리드) — Design

> PM 시안: 루트 `landing_page.pdf` (14p, untracked). PM 결정 2026-07-07 확정분 반영.
> www.inpa.kr 기존 랜딩은 **불변**. new.inpa.kr는 브랜드 경험용 별도 진입점.

## 0. PM 확정 결정 (locked, 2026-07-07)

1. **하이브리드 구성**: 시안 p2~p8 = 클릭 전환 시네마(검은 화면·타자기 연출) → 이후 화면이 밝아지며 일반 스크롤 랜딩으로 전환.
2. **시안 p1(파란 히어로) 제외**: 발표용 포스터였음. 영화는 p2(人波 정의)부터 시작.
3. **기능·가치 설명은 기존 www 랜딩 섹션 재사용**: 시안에 없는 기능 설명 구간은 `inpa_fe/app/page.tsx`의 섹션을 가져와 채운다.
4. **가격 공개**: 시안 p14의 4단 요금제(Basic 0원 / Manager 19,900 / Plus 19,900 / Super 39,900)와 카드 아래 부속 내용(부제 쿠폰 문구, 카드별 하단 설명, 가입자 FREE 블록)까지 표기. 가격 표기는 전사 VAT 규칙 준수: 노출 가격은 `19,900원 (VAT 별도)` 형식(최종 부가세 포함 금액은 결제 데스크에서만).
5. **입장 게이트 승인**: 첫 화면에서 "소리와 함께 시작 / 조용히 보기" 선택(브라우저 자동재생 정책 대응을 입장 연출로 소화). 건너뛰기 상시 제공.

연출 규칙:
- **타이핑**: 글자 단위, 글자마다 타자음(타닥). 커서 깜빡임. 문장부호 뒤 잠깐 쉬는 리듬.
- **타이핑 중 클릭** = 그 장면 문장 즉시 완성. **완성 후 클릭** = 다음 장면. 키보드 Space/→/Enter 동일.
- **상시 UI**: 우상단 "건너뛰기"(스크롤 파트로 즉시 이동) + 음소거 토글. 하단 진행 도트 + 타이핑 끝나면 "화면을 눌러 계속" 힌트.
- **prefers-reduced-motion**: 타이핑 생략, 문장 즉시 표시(소리도 기본 꺼짐).
- **소리**: WebAudio 합성 타자음(오디오 파일 없이, 톤 미세 랜덤화로 기계적 반복감 제거). 게이트에서 "조용히 보기" 선택 시 무음 진행.
- **이미지**(군중 p7, 업무 배경 p3/4): PDF에서 추출하거나 PM 원본 수급. 다음 장면 이미지 프리로드로 클릭 시 끊김 방지.
- 모바일 = 탭, 동일 경험.

## 3. 기술 설계

- **위치**: `inpa_fe` 저장소 내 신규 라우트(작업명 `app/new/`). 별도 프로젝트·저장소 없음.
- **도메인**: Vercel 프로젝트에 `new.inpa.kr` 추가 + **Next 16 `proxy.ts`**(구 middleware, Next 16에서 파일명·규약 변경됨)로 host가 `new.inpa.kr`인 요청만 해당 라우트로 rewrite. 구현 전 `node_modules/next/dist/docs/01-app/03-api-reference/03-file-conventions/proxy.md` 필독. www 트래픽 영향 0.
- **www 섹션 재사용**: `app/page.tsx`의 섹션 컴포넌트들이 파일 내 비공개 함수라, 공용 위치로 추출(export)하고 www 페이지는 동일 렌더 유지(프레젠테이션 변화 0 확인).
- **로그인 리다이렉트 없음**: www는 토큰 보유 시 /home으로 보내지만, new는 브랜드 경험 페이지라 로그인 상태여도 그대로 감상 가능.
- **SEO/OG**: 검색 본진은 www 유지. new는 색인 허용하되 전용 OG는 후속(PM 제작 이미지 연결 가능, `lib/public-og.ts` 빌더 재사용).
- **폰트**: S3~S4 타자기체 = 영문 모노스페이스 + 한글 대응(고정폭 느낌) 조합. 구현 시 확정.
- **DNS/콘솔 작업(PM)**: `new.inpa.kr` CNAME 추가 + Vercel Domains 등록. 클릭 단위 가이드는 구현 완료 시 제공.

## 4. Flags (후속 확인 필요)

1. **Manager 플랜 billing 등록 — PM 승인 완료(2026-07-07): 이번 범위에 포함.** 스펙: `code='manager'`, display_name 'Manager', **price_krw=19900 (VAT 별도)**, **한도는 Plus와 동일**(limit_ocr 200 / ai_compare 100 / analysis 200 / promotion 100), 설명 = 관리자(팀장·지점장·지사장) 전용, Plus 전체 기능 + 팀원 인사 관리·팀원 개별 실적 관리·팀 전체 실적 관리. `seed_billing`에 get_or_create 추가 + `PLAN_CODE` choices에 'manager' 추가(choices 정합용 마이그레이션 0006, DB 무영향) + admin 구독 부여 serializer choices에 'manager'. **주의: 팀 기능을 Manager 플랜 가입자만 쓰게 막는 권한 게이트는 이번 범위 밖**(현재 팀 기능은 동의 기반 무료) — 유료 전환 전 별도 결정.
2. 시안 부제의 "모바일 명함 무료 제작 쿠폰": 판촉물 디지털 자산으로 실제 이행 가능한지 운영 확인.
3. 시안 p14 가격에 VAT 표기가 없음 → 전사 규칙(2026-07-07 확정)에 따라 `(VAT 별도)` 소자 병기로 노출(시안과 유일한 의도적 차이).
