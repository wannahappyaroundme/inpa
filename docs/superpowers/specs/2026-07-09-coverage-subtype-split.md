# 담보 세분류 개별 인식 (PM 확정: 각각 별도로) — Spec

> 2026-07-09. PM 지시: 소액암/갑상선암(현재→유사암)·특정암(현재→일반암)·질병/고도후유장해(현재→상해후유장해)를 **각각 별도 담보로 인식**. 표준트리 leaf는 이미 존재 → §8 트리 확장 아님(순수 라우팅). 사망 수정(2026-07-09, DeathCoverageRoutingTests)과 동형. TDD + 골든 앵커로 잠금.

## 배경 (탐사 완료)
- 대상 leaf 존재 확인: 소액암진단비·특정암진단비·질병후유장해·고도후유장해(O). 갑상선암은 별도 leaf 확인 필요(없으면 §8 → 제외하고 보고).
- 현재 오분류(골든 failures): 소액암진단급여금→유사암(기대 소액암진단비), 갑상선암→유사암, 특정암→일반암, 질병후유장해(3~100%)→상해후유장해, 상해80%이상후유장해/고도후유장해(80%이상)→상해후유장해(기대 고도후유장해).
- 매처: `ocrparsing.py::COVERAGE_KEYWORDS`(단일 소스, claude_parser가 import). `coverage_bridge.py::PARSER_TO_STD`(파서 path→표준 leaf). 골든 채점기 `analysis/golden_eval.py`(prod 매칭 순서 재현) + `data/golden_set.json` 앵커.

## 설계 (마이그레이션 0, 데이터/코드만)

각 세분류마다: (1) `PARSER_TO_STD`에 파서 path→표준 leaf 매핑 추가(leaf 존재 확인), (2) `COVERAGE_KEYWORDS`에서 그 세분류 키워드를 상위 경로(유사암/일반암/상해후유장애)에서 **떼어내 전용 경로로 이동**, (3) 골든 앵커 추가(정답 잠금), (4) 매처 직접호출 회귀 테스트.

### 대상별
1. **소액암** → `진단비->암->소액암` (PARSER_TO_STD (진단비,암,소액암)→소액암진단비). 유사암 경로에서 '소액암' 키워드 이동. 소액암 계열 별칭(소액암진단급여금 등) 라우팅 확인.
2. **갑상선암** → leaf 존재 시 `진단비->암->갑상선암`. 없으면 §8이므로 **이번 스코프 제외 + 보고**(유사암 유지).
3. **특정암** → `진단비->암->특정암` (leaf 특정암진단비 존재). 일반암 경로에서 '특정암진단'/'특정암' 키워드 이동. ★ 트랩: '특정(소액)암'은 괄호로 연속성 깨짐 → 이번엔 괄호 없는 '특정암' 우선(괄호 케이스는 별도 노트).
4. **질병후유장해** → `상해->질병->질병후유장해`(또는 적절 sub). 상해후유장애 경로의 '질병후유장해','질병80%이상후유장해','질병후유장해(3~100%)' 등을 이동.
5. **고도후유장해** → `상해->상해->고도후유장해`. 상해후유장애 경로의 '고도후유장해','상해80%이상후유장해','후유장해(80%이상)','고도후유장해(80%이상)' 등 **고도/80%이상 계열만** 이동. ★ 트랩: '상해후유장해'(일반)와 '상해80%이상후유장해'(고도) 구분 — 80%/고도 토큰 있는 것만 고도로, 나머지는 상해후유장해 유지. longest-first로 구체 키워드가 이기게.

### ★ §7 substring 트랩 필수 회피 (사망 수정 교훈)
- 세분류 키워드가 상위 키워드보다 **길어야** longest-first로 이긴다. 필요시 구체 복합어 추가.
- **회귀 감시 앵커**: 일반암진단급여금→일반암, 유사암진단급여금→유사암, 상해후유장해급여금→상해후유장해(일반)가 세분류 이동으로 안 깨지는지 반드시 앵커로 고정.
- 각 이동 후 **골든 정확도가 오르고 앵커 100% + 회귀 앵커 유지**를 확인. 하나라도 기존 매핑을 깨면 그 세분류는 재검토.

### 검증 (필수)
- `_match_by_keywords` 직접 호출 회귀 테스트(세분류 라우팅 + 상위 무변경 고정) — 사망 패턴.
- 골든 정확도 재측정(0.6569 → 상승 기대) + 앵커 전원 통과 + 기준선(GOLDEN_SET_MIN_ACCURACY) 실측 바로 아래로 상향.
- **전체 BE 스위트**(inpa.analysis + inpa.insurances 필수 — 입원일당 때처럼 기존 회귀 테스트가 의도적 설계를 지킬 수 있음. 깨지면 그 세분류 이동이 기존 설계와 충돌 = 재검토/제외).
- FE 무변경(BE-only). 마이그레이션 0.
