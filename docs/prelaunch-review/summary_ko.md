# 인파(Inpa) 출시 전 리뷰 · 경영 요약

> 2026-07-04 · 절차: 코드 전수 감사(6영역 병렬 + 교차 검증) → 22인 페르소나 3라운드 토론(제안 88건) → 정량 채점(22인 × 30개 기능 × 5차원 = 660표). 원자료: project_audit.md · debate.md · features.md · scores.csv (모두 docs/prelaunch-review/).

## 1. 한눈에

- 제품 완성도는 프리런치 단계 기준 상위권입니다. 설계사 업무 루프(발굴 → 분석 → 비교 → 예약 → 관리)가 끝까지 실동작하고, 법적 가드(비교안내서 발행 차단·국외이전 동의 게이트·병력 차단·기준선 중립)는 코드에서 전부 확인됐습니다. 고객 데이터 격리는 뷰 약 100개 전수 점검에서 구멍 0건.
- 다만 "보이지 않는 곳"이 약합니다: 배포 때마다 분석 데이터가 초기화되는 버그(LB-1), 잘못된 문구로 받는 동의(LB-2), 실서버의 데모 데이터(LB-3), 울리지 않는 알림, 모니터링·백업 부재, 콜드스타트.
- 대표 결정 반영: 출시 차단급+높은 위험 전량 수정 확약, 결제는 계좌이체 → KICC PG(이지페이), 사업자 보유(법정 표시 기재 가능), 자체 서버(백업 활용).

## 2. 핵심 결정 (토론 합의)

1. **출시 게이트 = 13개 묶음** (아래 3번). 이 묶음이 끝나기 전에는 공개 홍보를 하지 않는다.
2. **유료화는 초대제 정가 파일럿(Founding 20)으로 시작.** 무료층을 없애는 유료 단독 출시는 기각. 계좌이체 결제 데스크(가격 확정+세금계산서 절차)가 선행 조건.
3. **실물이 없는 화면은 채워질 때까지 숨긴다.** 이미지 없는 판촉물 카탈로그, 울리지 않는 알림 설정은 신뢰를 깎는다.
4. **정규화 사전이 유일하게 축적되는 자산.** 골든셋 평가(CI)와 비용·품질 텔레메트리를 출시 주간에 깐다.
5. **법적 거부권 행사 0건.** 위험 아이디어(관리자 승인 큐, 쿠폰 후기, 유료 단독)는 토론에서 이미 탈락. 남은 30개는 전부 법적 안전 5점 이상.

## 3. 출시 차단 이슈 13 (패널 합의 실행 순서)

1. **동의 문구 단일 소스화 + 재동의** (가중 4.22점, 순위 24/30)
2. **시드 안전화(자연키 upsert) + 부팅 분리** (가중 6.71점, 순위 5/30)
3. **고객 링크 상시 가동 백엔드** (가중 7.37점, 순위 3/30)
4. **야간 암호화 백업 + 복구 리허설** (가중 4.29점, 순위 23/30)
5. **Sentry + 합성 모니터링** (가중 4.47점, 순위 22/30)
6. **단일 배치 러너 + 생존 신호** (가중 4.79점, 순위 18/30)
7. **리마인더 발송 + 아침 8시 다이제스트** (가중 7.72점, 순위 2/30)
8. **신뢰 대청소(데모·플레이스홀더·유령 화면)** (가중 5.55점, 순위 10/30)
9. **PII 로그 정리 + 법정 표시사항** (가중 3.88점, 순위 26/30)
10. **동의 철회 + 보존기간 자동 삭제** (가중 3.68점, 순위 28/30)
11. **계좌이체 결제 데스크 + 가격 확정** (가중 5.39점, 순위 12/30)
12. **/s 죽은 버튼 수정(전화+콜백)** (가중 6.51점, 순위 8/30)
13. **메리츠 데이터 저장소 정리(1단계)** (가중 2.68점, 순위 30/30)

주의: 이 13개는 점수가 낮아도 뺄 수 없는 확약 항목입니다. 채점 공식이 설계사 가치 45%라서 위험 제거형 작업은 구조적으로 점수가 낮게 나옵니다. 점수는 순서 참고용입니다.

## 5. 출시 후 로드맵 권고

- **MVP(출시 주간, 7개):** 오늘 전화 리스트 · 초대제 정가 유료 파일럿(Founding 20) · 카톡 링크 미리보기 카드 · 팀 초대 링크 · 지점 멀티시트 코드 결제 · 권유 단어 자동 차단(CI+서버) · 화면 상태 정리(무음 실패 제거)
- **Phase 2(출시 후 1~2개월, 8개):** 유지율(13/25회차) 모듈 · 비교 스냅샷 불변 보관 · 커뮤니티 담보 사전 피드백 · 토큰 페이지 로딩 스켈레톤 · 정규화 골든셋 평가(CI) · 판촉물 실사 이미지 · 추천인 쿠폰 루프 · Claude 비용·파싱품질 텔레메트리
- **Backlog(2개):** 활성화 퍼널 계측 + UTM · SEO 기본기(robots·sitemap) (점수는 낮지만 각각 하루짜리 작업이라 틈나면 처리 권장)
