# 06 · 비즈니스 모델 · 가격 · 유닛이코노믹스

> **인파(Inpa)** — 위촉직 보험설계사를 위한 AI 영업 파트너의 과금 설계서.
> **읽는 사람:** 투자자(단위경제 검증) · 디자이너(업그레이드 모달·과금 UX) · 개발자(크레딧 게이트·402 핸드오프).
> **전제:** 아래 모든 숫자 중 협회 공시·Claude API 공시단가가 아닌 항목은 전부 **(추정)** 이다. 베타 90일 실측 전까지 가설로 취급하며, 한도·전환율·LTV는 토큰화(코드 상수화) 금지 — 측정 후 확정한다.
> **정식 출시일 미확정** (`SUPER_BETA_EXPIRY=2099-12-31`). 베타 기간은 `FREE_TIER_UNLIMITED=True`로 모든 한도·워터마크를 우회한다.

## 0. 한 장 요약 (TL;DR)

```
가치 원칙   :  "분석은 풀고, 행동은 막는다"
              분석(OCR·히트맵·비교안내서 1건 생성) = 무료 미끼
              행동(발송·복수생성·AI메시지·워치독·귀속) = Plus+ 과금

4-tier      :  Basic 0원 / Plus 19,900원 / Premium 89,900원 / Super 견적

흑자 근거   :  Claude 호출당 원가를 캐싱+Batches로 수백 원대까지 압축
              → 개인이 월 2만 원 내는 AI 영업비서가 단위경제상 흑자 진입

단위경제(추정): ARPU 25,000 / LTV 60만 / CAC 3~5만 / LTV-CAC 12~20x

미해결      :  Plus↔Premium 갭 4.5배(신입 핵심기능 Premium 편중) → 베타 후 재검토
```

## 1. Freemium 4-tier (확정)

기능은 **전부 열되**, 무료는 **월 횟수 제한(`monthly_credit`)** 으로 묶는다. 핵심 레버는 가격이 아니라 **"무료 한도를 어느 *행동*에 긋느냐"** 다. 분석은 무료로 풀어 wedge(갈아타기 비교안내서)를 한 번 보여주고, **반복 발굴·발송·자동화·귀속에서 과금**한다.

| Tier | 가격(월, 추정) | 무료/유료 게이트 라인 | 핵심 타겟 |
|---|---|---|---|
| **Basic** | **0원** | 증권 OCR · 보장 히트맵 · 갈아타기 비교안내서 **1건 생성**까지. 컴플라이언스 가드레일 **전부 무료**(위생) | 신입 콜드스타트 · 체험 |
| **Plus** | **19,900원** | 비교안내서 **발송·복수 생성** · AI 카톡 메시지(클립보드) · OCR 한도 상향 · 진단 무제한 | 발굴 절박 신입~중견 (커피 4잔 체감가) |
| **Premium** | **89,900원** | 만기·갱신 **워치독** · 추천코드 **귀속/리퍼럴** · DB 보장공백 **전체 배치 스캐너** · 알림톡(심의 후) | 리텐션 금광 중견 (DB스캐너 전용가) |
| **Super** | **견적** | 팀/지점 단위 시트 · 관리자 대시보드 · 다중 설계사 관리 | GA 지점장 · 팀 |

### 무료 한도 후보 라인 (추정 — 베타 90일 실측 후 확정)

| 행동 | Basic(무료) | Plus | Premium | Super | 차감 크레딧 종류 |
|---|---|---|---|---|---|
| 증권 OCR 등록 | 월 10건 (추정) | 월 100건 (추정) | 월 500건 (추정) | ∞ | `insurance_credit` |
| 고객 분석/히트맵 | 월 3건 (추정) | 월 20건 (추정) | 월 100건 (추정) | ∞ | `ai_credit` |
| 비교안내서 **생성** | **1건 체험** | 무제한 | 무제한 | ∞ | `ai_credit` |
| 비교안내서 **발송**(공유링크) | **0건 (잠금)** | 허용 | 허용 | 허용 | — (게이트) |
| AI 카톡 메시지 생성 | **잠금** | 허용 | 허용 | 허용 | `ai_credit` |
| 워치독 · 귀속 · 리퍼럴 | **잠금** | 잠금 | 허용 | 허용 | — (게이트) |
| 컴플라이언스 가드레일 | ✅ 무료 | ✅ | ✅ | ✅ | — (전 티어 위생) |

> **워터마크 강제 과금 금지** — 가드레일(§97 면책·정확요건)은 **전 티어 공통**이어야 한다. "돈 안 내면 위험한 비교안내서를 내보낸다"는 구조는 §97 방패의 신뢰를 깨므로 절대 금지. 워터마크를 떼는 대가로 과금하지 않는다. 진짜 바이럴 수익원은 **추천코드 귀속**이다.

### 과금 거버넌스 (개발 핸드오프)

```
402 응답 shape (크레딧 소진):
  { reason: 'CREDIT_EXHAUSTED', limit, remaining }
  → FE: UpgradeGuideModal 노출

베타 우회:
  FREE_TIER_UNLIMITED = True   (settings/base.py)
  → 한도·워터마크 전부 무시. 정식 출시 시 False flip.

크레딧 sentinel:
  monthly_*_credit = 0  →  "무제한"(Super + 베타), 헤더는 ∞ 표기
  remaining == null / is_unlimited == true 로 판별 (remaining==0 아님)
```

## 2. 비용구조 — Claude API (흑자의 단단한 근거)

### 검증 단가 (per MTok, 2026-06 공시)

| 모델 | 입력 | 출력 | 용도 |
|---|---|---|---|
| `claude-opus-4-8` | **$5** | **$25** | M1 갈아타기 비교안내서 · 정규화 사전 학습 (정확도 critical, §97 합법성) |
| `claude-sonnet-4-6` | $3 | $15 | (예비) 중간 난이도 |
| `claude-haiku-4-5` | **$1** | **$5** | M2~M4 · M6 다건 OCR (고빈도 · 저위험) |

> 모델별 토큰 측정은 **`messages.count_tokens`** 로 사전 측정한다. `tiktoken` 금지 — OpenAI 토크나이저라 Claude 토큰을 15~20% 과소계산하며, Opus 4.8/Haiku는 토크나이저가 다르다.

### 원가 절감 2단 레버 — per-call 원가를 수백 원대로

**① Prompt caching (정규화 사전을 system 블록에 고정)**

보험사별 담보명 정규화 사전(NormalizationDict)은 모든 비교안내서 호출이 공유하는 거대 고정 프리픽스다. 이를 system 블록 맨 앞에 두고 끝에 `cache_control: {type: "ephemeral"}` breakpoint를 찍으면:

- **캐시 읽기 ≈ 입력단가의 0.1×** / 쓰기 ≈ 1.25×(5분 TTL) / 2×(1시간 TTL)
- **5분 TTL 기준 2회 호출이면 손익분기** (쓰기 1.25× + 읽기 0.1× = 1.35× < 미캐싱 2×)
- **고객별 가변값(병력·담보 입력)은 반드시 breakpoint *뒤*에** 둔다 — 안 그러면 고객마다 사전 전체를 재처리한다(silent invalidator).
- 검증: 반복 요청에서 `usage.cache_read_input_tokens`가 0이면 무효화 요인(타임스탬프·UUID·정렬 안 된 JSON) 존재.

**② Batches API (야간 배치는 50% 할인)**

- M2 보장공백 스캐너(야간 전체 DB 배치) · M4 만기 워치독 문안 = latency 비민감 → **표준단가 50% 할인** (최대 100,000 요청/배치).
- 단 Batches는 prompt caching·서버 폴백 제한 → **M1 비교안내서는 실시간 경로 유지** (Opus + 캐싱).

### per-call 원가 시산 (추정)

| 호출 | 모델 | 경로 | 호출당 원가(추정) |
|---|---|---|---|
| 비교안내서 1건 (Opus, 캐시 hit) | opus-4-8 | 실시간 + 캐싱 | 수백 원대 ($0.1~0.3) |
| 다건 OCR 1건 | haiku-4-5 | 실시간/배치 | 수십 원대 ($0.02~0.05) |
| 야간 DB 스캐너 (배치) | haiku-4-5 | Batches 50%↓ | 1건당 ~수 원 |

> **"왜 지금"의 흑자 논거:** 캐싱·Batches로 1인 설계사의 월 AI 원가가 수백~수천 원대로 내려왔다. 개인이 월 19,900원 내는 AI 영업비서가 **비로소 흑자 단위경제에 진입**한다. 이 타이밍이 진입 명분이다.

### 비용 거버넌스 (설계 원칙)

- [ ] **호출당 비용 로깅** — 모델·토큰·캐시 hit율을 호출마다 기록
- [ ] **월 예산 캡 + 임박 알림** — 80% 도달 시 운영자 경보
- [ ] **North Star 보조지표** = **"발송 액션당 AI 비용"** (단순 호출 수가 아니라 매출로 이어지는 발송 1건당 비용)
- [ ] **재시도 정책** — 429/5xx는 SDK 자동 백오프(max_retries=2), **401/403/400은 재시도 금지**

## 3. 유닛이코노믹스 (LTV / CAC, 추정)

> **투자자 서사의 중심은 절대 시장가치(ARR)가 아니라 단위경제다.** 시장 규모(TAM 38만 → SAM 12만 → SOM 2,400~3,600명, ARR 7.2~10.8억)는 02 문서를 참조. 여기서는 한 명의 설계사가 흑자인가만 본다.

| 지표 | 값(추정) | 산출 논리 |
|---|---|---|
| **ARPU** | 약 **25,000원/월** | Plus 19,900 · Premium 89,900의 가중평균 가정값 (실제 믹스 미검증) |
| **LTV** | 약 **60만원** | 24개월 × ARPU. **quota=생존**인 직군이라 일반 SaaS보다 리텐션이 구조적으로 높다는 가설 |
| **CAC** | 약 **3~5만원** | 지점 군집 1:1 영업(동료 5~10명 노출로 한계비용 분산) + 추천코드 바이럴 귀속 |
| **LTV / CAC** | **12~20x** | 펀딩 서사 성립 임계선 (일반 SaaS 3x 기준의 4~6배) |

### LTV 근거 — quota=생존 리텐션 가설

```
위촉직 = 매달 quota 압박 → 발굴이 멈추면 생계가 멈춤
→ 발굴 도구를 한 번 영업 루프에 끼우면 끊기 어려움 (해지=발굴 포기)
→ 일반 SaaS(이탈 자유)보다 24개월 잔존이 높다는 가설
※ 검증 대상: 베타 90일 4주 잔존율 + Plus→Premium 전환
```

### CAC 근거 — 지점 군집 + 추천 루프

door-to-door는 표면상 비싸 보이지만, **지점 단위 군집 노출**(지점장 데모 1회 → 동료 5~10명 동시 노출)과 **추천 더블사이드 루프**가 한 명 전환의 한계비용을 빠르게 떨어뜨린다. 채널 우선순위는 CAC 낮은 순 — 인바운드 SEO → 지인·소개 바이럴 → 지점 군집 door-to-door (상세는 07 문서).

### 전환 퍼널 가설 (베타 검증 대상)

```
분석 무료 사용
   │  (aha: 5분 안에 증권→히트맵→비교안내서)
   ▼
업그레이드 모달 (비교안내서 발송 시도 / 크레딧 소진 402)
   │
   ├──▶ Plus 전환    8% (추정)
   └──▶ Premium 전환 3% (추정)

가속 장치: 14일 풀언락 트라이얼 → aha를 먼저 경험시키고 과금
```

## 4. 앵커링 · 과금 UX (디자이너 핸드오프)

업그레이드 모달은 **가격 닻을 높게 던지고 Plus로 회수**하는 구조다.

```
┌──────────── 업그레이드 모달 ────────────┐
│                                          │
│   Premium  89,900원/월                   │ ← 먼저 노출(앵커: 높은 값)
│     · 워치독 · 귀속 · DB 전체 스캐너      │
│                                          │
│   ┌────────────────────────────────┐    │
│   │  Plus  19,900원/월  ★ 가장 인기  │    │ ← 시각 강조 → 상대적 저렴 인식
│   │    · 비교안내서 발송 · AI 메시지  │    │
│   └────────────────────────────────┘    │
│                                          │
│   연결제 시 2개월 무료 (LTV 방어)        │ ← 연납 유도 = 이탈 방어
│                                          │
└──────────────────────────────────────────┘
```

- **앵커:** Premium 89,900 먼저 → Plus를 "가장 인기"로 표기 → 상대적 저렴함 부각.
- **연결제 2개월 무료:** 연납 LTV를 묶어 이탈을 구조적으로 방어.
- **트리거:** 402 `CREDIT_EXHAUSTED` 응답 또는 잠긴 행동(발송·AI메시지) 클릭 시 모달 진입.
- **정직성 레드라인:** 모달·버튼 어디에도 "원탭 자동발송"·"심의 안전/완료 배지" 카피 금지. 발송은 **클립보드 복사**로 정직하게, AI 결과물은 **"AI 1차 보조, 최종책임 설계사"** 면책 고정.

## 5. 리퍼럴 보상 거버넌스

귀속 인프라(`share_token?ref=<설계사코드>` + 이벤트 계측)는 **Day1에 깐다** (사후 복원 불가). 단 바이럴 *루프 활성화*는 Phase2 (계측은 1일차, 캠페인은 2단계).

| 항목 | 규칙 |
|---|---|
| 보상 형태 | 추천 1명 가입당 **Premium 1개월** 더블사이드(추천인+피추천인 양쪽) |
| **상한** | **연 3개월** (무료 헤비기능 무한 노출로 전환율 갉아먹기 방지) |
| 셀프추천 차단 | 본인 추천코드 자기 사용 불가 |
| 유령계정 차단 | 동일 기기/IP 반복 가입 차단 + 최소 활동(증권 1건 등록 등) 후 보상 확정 |
| 정산 시점 | 피추천인 onboarding 완료(첫 분석) 후 귀속 확정 — `referral_attributed` 이벤트 |

> 상한·차단 룰은 **과금 거버넌스와 사전 합의 완료**. 자동매핑/자동승격 임계(hit_count 5회+ 자동 허용 여부)는 운영 주체 미정으로 openQuestion에 남김.

### 비용 측면 손익분기 (추정)

```
변동비(AI)   : 발송/분석 1건당 수백 원대 (캐싱+Batches로 압축)
             → ARPU 25,000원 대비 AI 변동비 비중 낮음(추정 5~15%)
             → gross margin 구조적으로 높음

고정비       : 인건비 + IDC 서버(foliio 공유) + Sentry/GA4 등
             → MRR 손익분기점은 고정비 규모에 종속 (현 단계 미확정)

핵심         : 단위경제(LTV-CAC 12~20x)가 성립하면
             고정비 회수는 유료 설계사 수의 함수.
             기준 시나리오 M12 MRR 2,000만원이 1차 BEP 가늠선(추정).
```

> **BEP 정밀 산출은 고정비(인건비·운영비) 확정 후.** 현 단계에서 확정 가능한 것은 *변동비 흑자*(호출당 원가 < 행동당 과금)뿐이며, 이것이 펀딩 서사의 핵심이다.
