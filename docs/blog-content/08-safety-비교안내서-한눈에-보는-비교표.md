## 본문 (body)

고객이 "이거 갈아타는 게 나아요?"라고 물으면, 말로만 설명하기가 참 어렵습니다. 지금 든 보험이랑 새로 권하는 보험을 종이 한 장에 나란히 놓아 보면, 그 대화가 훨씬 쉬워집니다.

비교안내서는 지금 가입한 보험과 새로 권하는 보험을 한 장에 나란히 정리한 표입니다. 담보가 뭐가 늘고 줄었는지, 보험료는 얼마나 달라지는지, 그리고 갈아탈 때 손해 보는 부분(보험을 해지하면 돌려받는 돈, 새 보험에서 처음 얼마간 보장이 제한되는 기간 등)까지 한자리에 담습니다. 좋은 점만 듣고 정하지 않도록 아쉬운 점도 같은 표에 넣는 것이 핵심입니다. 그래야 고객이 직접 보고 스스로 정할 수 있습니다.

## 비교안내서가 왜 필요한가요?

보험을 갈아타면 보장이 넓어지는 것처럼 보여도, 알고 보면 지금 보험을 해지하며 손해를 보거나 새 보험은 가입 초기에 보장이 제한될 수 있습니다. 이런 부분을 빼놓고 좋은 점만 이야기하면, 나중에 고객이 "그런 얘긴 못 들었다"고 할 수 있습니다.

한 장에 좋은 점과 아쉬운 점을 같이 적어 두면, 고객은 충분히 알고 정하게 되고 설계사님은 "제대로 안내했다"는 근거를 남기게 됩니다. 갈아탈 때 고객이 불리한 점을 알고 정하도록 돕는 것은 관련 규정에서도 중요하게 봅니다.

## 한 장에 무엇을 나란히 담아야 하나요?

지금 보험과 새 보험을 아래 항목대로 왼쪽·오른쪽에 놓고 비교하면 고객이 한눈에 이해합니다.

| 비교 항목 | 지금 보험 | 새로 권하는 보험 |
| --- | --- | --- |
| 담보 이름과 보장 금액 | (담보별로) | (담보별로) |
| 월 보험료 | | |
| 보장 기간, 납입 기간 | | |
| 빠지거나 줄어드는 담보 | | |
| 해지하면 돌려받는 돈 | | |
| 처음 보장이 제한되는 기간 | | |

특히 담보는 회사마다 이름을 다르게 부르는 경우가 많습니다. 같은 보장인데 이름만 다른 것을 같은 줄에 맞춰 놓아야, 고객이 "이게 이거랑 같은 거구나" 하고 헷갈리지 않습니다.

## 갈아탈 때 손해 보는 부분은 어떻게 알려주나요?

좋은 점 옆에 아쉬운 점을 꼭 같이 적습니다. 아래는 자주 빠뜨리는 항목입니다.

- 지금 보험을 해지하면 돌려받는 돈이 그동안 낸 돈보다 적을 수 있습니다.
- 새 보험은 가입하고 처음 얼마간 일부 보장이 안 되거나 적게 나올 수 있습니다.
- 나이가 올라가서 같은 보장인데 보험료가 더 비싸질 수 있습니다.
- 지금 건강 상태 때문에 새 보험은 일부 담보가 빠지거나 가입이 어려울 수 있습니다.

이런 내용은 고객이 갈아타기로 정하기 전에 따로 분명히 짚어 주는 것이 좋습니다. 표에 한 줄로 적더라도, 말로 한 번 더 확인해 주세요.

## 고객이 보고 스스로 정하게 하려면?

"이게 낫다"고 정해서 밀어붙이기보다, 표를 같이 보면서 "이 부분은 좋아지고 이 부분은 아쉬운데 어떻게 생각하세요?"라고 물어보는 편이 좋습니다. 고객이 스스로 내린 결정은 나중에 마음이 바뀌는 일이 적고, 신뢰도 더 쌓입니다.

## 인파는 이 표를 어떻게 도와주나요?

인파에 지금 보험 증권과 새 제안을 올리면, 담보를 자동으로 표로 정리해 지금 보험과 새 보험을 나란히 놓아 줍니다. 회사마다 다르게 부르는 담보 이름도 같은 줄로 맞춰 주고, 담보가 넉넉한지 부족한지 신호등처럼 색으로 보여 줍니다. 어느 쪽이 나은지 판단과 고객 안내는 설계사님 몫이고, 인파는 비교표를 손에 쥐여 드리는 데까지 돕습니다.

인파는 보험을 중개·권유하지 않는 분석·정리 소프트웨어입니다. 보장 판단과 고객 안내는 설계사님의 업무입니다.
