# 인파 노트 — 초석 원고 (첫 10편)

브랜드 보이스 가이드([00-brand-voice-guide.md](00-brand-voice-guide.md)) 기준으로 쓴 첫 10편 초안입니다.

## 쓰는 법
1. `www.inpa.kr/admin/blog` → **새 글**
2. 각 파일 상단 메타(제목·카테고리·요약·태그·SEO·슬러그)를 해당 칸에 입력
3. **본문(body)** 을 본문 칸에 붙여넣기 → 미리보기 확인 → **게시**
4. 대표 이미지(커버)는 흰 배경 + 카테고리색 옅은 틴트 + 제목 타이포로(가이드 §6). 없어도 발행 가능.

## ⚠️ 발행 전 필수 확인
- **안심 가이드(safety) 3편은 실변호사 1회 검토 후 발행**: 07(부당승환)·08(비교안내서)·10(갈아타기 유의). 조문·기간·과태료 숫자는 국가법령정보센터(law.go.kr)·금융감독원 원문으로 재확인.
- 어느 글이든 **특정 보험사·상품 실명 추천/비교 금지**(교육 콘텐츠 선 유지).
- 바이라인(작성자/감수자 실명)을 넣으면 검색 신뢰(E-E-A-T)에 유리.
