# Foliio 영업지원 피벗 — 기획 패키지

> **가치 전환**: "상위 1% 설계사를 위한 정밀 분석 계산기" → **"보험설계사 영업지원 AI 운영체제"**
> 5개 페르소나(기획·디자인·마케팅·개발·영업전문가) 멀티에이전트 토론 → 수렴한 종합 기획.
> 작성 2026-06-17. 이 폴더가 새 제품 방향의 시작점.

---

## 0. 한 문장 비전

> **보험설계사가 "고객을 만나기 전 3분, 만난 후 1분, 그리고 잊지 않아야 할 모든 순간"을 AI가 대신 준비해주는 — 분석을 무기로, 콘텐츠를 미끼로, 리텐션을 복리로 바꾸는 한국형 영업지원 운영체제.**

세 단어: **분석 = 무기**, **콘텐츠 = 미끼**, **리텐션 = 복리**.

> 🔑 **발굴-우선 보정 (2026-06-17, PM 현장정보 반영 → [문서 05](05-lead-acquisition-by-product.md))**: 설계사는 매달 quota 압박으로 **신규 발굴이 진짜 병목**이다(설득 < 발굴). 한국 발굴의 현실 1위는 콘텐츠 인바운드가 아니라 **DB영업 = 기존 가입자 갈아타기(승환)** + 지인·소개. Foliio의 **기존vs제안 비교 엔진이 이미 그 갈아타기 무기**이고, **부당승환(보험업법 §97) 규제를 '자동 비교안내서'라는 합법 방패 기능으로 뒤집는 것**이 최대 차별점. 그래서 발굴 기능(갈아타기 비교안내서 + 기존고객 공백 스캐너)을 **MVP로 승격**한다.

> 🔑 **분석 깊이 보정 (2026-06-17 → [문서 06](06-coverage-taxonomy-reference.md))**: 현 분석은 **7-factor(7개 보장 카테고리)** 만 보여주지만, 실제로는 **생명·손해 100개+ 담보 전체 체계**가 *한눈에* 보이는 분석·영업·시각화 자료여야 한다. 같은 담보도 보험사별 이름이 달라(예: 일반암 진단비 = 삼성 *암진단급부금* / 교보 *암진단보험금*) **담보명 정규화 사전**이 분석·OCR·갈아타기 비교의 전제. 핵심 통찰: **설계사는 분석을 직접 하지만 '틀'이 없다** → Foliio가 전체 담보 체계를 '틀'로 제공하는 것이 해자.

## 1. 무엇이 바뀌나 (From → To)

| | 기존 (분석 계산기) | 피벗 후 (영업지원 OS) |
|---|---|---|
| 핵심 가치 | 증권 넣으면 8케이스 보험료 계산·보장분석표 출력 | 같은 분석을 **영업 행동의 시작점**으로 재배치 |
| 분석의 위치 | **끝점** — 설계사 혼자 보는 정적 자료 | **트리거** — 카톡 멘트·후속 큐·리드 루프로 변환 |
| 설계사 노동 | 전달·후속·재활용 전부 수기 | 준비→접촉→유지→소개를 **거의 무노동**으로 |
| KPI | 분석 건수 | **미팅당 공유링크 발송수 × 고객 열람률 × 소개 귀속 리드수** |

**피벗의 본질** = 끝점을 시작점으로. 이미 깔린 `증권 OCR → 8케이스 → 공유링크` 파이프라인 위에 ①컴플라이언스 안전망 ②만기·생일 자동 후속 큐 ③바이럴 귀속 루프를 얹는다. **새로 만드는 게 아니라 가진 자산을 재배치**한다.

## 2. 5대(+1) 필러

| 필러 | 한 줄 정의 |
|---|---|
| ① **분석-콘텐츠 파이프라인** | OCR→8케이스→공유링크를 '자료 생산'이 아닌 '접촉 트리거'로. 경쟁사가 못 잇는 **유일한 진짜 해자**. |
| ② **컴플라이언스 안전망** | 모든 AI 텍스트가 단일 게이트 통과(정규식 차단 + LLM warn + 사람 승인). 해자 아닌 **위생 + 면책**. |
| ③ **리텐션 엔진** | 만기·생일 데이터 + cron + 알림 인프라로 실효/해지 방어. ROI 최상 구간. |
| ④ **리드 성장 루프** | 공유링크 하단 추천코드 → 고객의 지인을 `referred_by`로 자동 귀속. **도태 1순위 = 리드 고갈** 정면 공략. |
| ⑤ **퍼스널 브랜딩 콘텐츠** | promotion 14종 + 성과 KPI 재활용. 보장·수익률·비교 원천 배제로 **광고심의 회피**. 후행 활성화. |
| ⑥ **측정·과금 거버넌스** | 북극성 지표 + AI 호출을 크레딧에 게이팅해 비용 누수 차단. |

## 3. MVP 범위 (7개 — 좁게)

전부 **기존 자산 재활용 비중이 높고 임팩트가 큰** 것만 골랐다.

- [ ] **사전준비형 원클릭 니즈환기** (Analysis→Kakao) — 분석 JSON → 카톡 멘트 3종 + 공유링크. *재활용 90%*
- [ ] **AI 보장공백 진단 + 상담 내러티브** — 분석탭 위에 진단 카드 + 화법. *숨은 폭탄: 표준 보장 기준선 정의*
- [ ] **보험 컴플라이언스 가드레일** — 공유 검열 미들웨어. *이게 없으면 다른 AI 기능 전부 출시 불가*
- [ ] **만기·갱신 자동 워치독** — cron이 케어 타이밍 + 갱신 시뮬 멘트를 '오늘의 케어' 큐로
- [ ] **AI 카톡 메시지 엔진** — 목적 enum(첫인사/안부/약속/생일/만기/소개요청/거절후속) + 가드레일. *5개 기능의 공용 엔진*
- [ ] **증권 다건 일괄 OCR + 매핑 정확도 강화**
- [ ] **PII 동의·민감정보 국외이전 동의 동선** — *법무 선결, AI 기능 전체의 출시 가부를 결정*

기능 분포: **MVP 7 · Phase2 7 · Phase3 3 · Later 2** (총 19개)

## 4. 비즈니스 모델 — Freemium (2026-06-17 확정)

> **방향 확정 (PM)**: 구독 게이팅 우선이 아니라 **Freemium**. 모든 기능을 *써볼 수는 있되* 무료는 **횟수 제한**, 더 많이/자주 쓰는 헤비유저만 구독으로 전환. **AI 기업(ChatGPT·Claude 등)의 표준 모델.** (DB 명단 판매·소싱은 우리가 합법적으로 구할 방법이 없으므로 비즈니스에서 제외.)

핵심 원칙: **기능은 열되 '사용량'을 과금** — 무료로 가치를 충분히 체험시키고, 실적이 돈이 되는 헤비유저가 자연 전환.

| 구분 | 과금 레버 | 내용 |
|---|---|---|
| **Free (횟수 제한)** | 월 N회 한도 | 증권 분석·갈아타기 비교안내서·AI 메시지·콘텐츠를 **전부 써보되 월 사용 횟수 캡**. 가치 체험 + 바이럴 연료 |
| **Pro (구독)** | 한도 대폭 상향 | 분석·발송·자동화 무제한급 + 리텐션 워치독 + 추천코드 귀속 + 성과 브랜딩 (**매일 켜는 헤비유저**) |
| **Team (견적)** | 좌석/지점 | 팀·지점 일괄 + 관리자 대시보드 + 팀 바이럴 |

- 무료/유료 경계 = **횟수(credit)**. 현 `monthly_credit` 인프라 그대로 활용 → 비용 누수 차단 + Freemium 게이트.
- 가격(예: Pro 19,900~)은 베타 데이터로 전환율 보고 확정 — **티어 수보다 "무료 한도를 어디에 긋느냐"가 핵심 레버**.
- 컴플라이언스 안전망은 **전 티어 공통**(과금 차별화 X). **워터마크 강제 과금 금지**(이탈 유발).

## 5. 반드시 결정해야 할 것 (PM 의사결정 대기)

가장 무거운 리스크 = **법무/컴플라이언스가 AI 기능 전체의 출시 가부를 쥐고 있음**.

1. **표준 보장 기준선을 누구 권위로 정의하나?** (금감원/보험연구원/자체) — 진단 기능 출시 선결. *숨은 EFFORT 폭탄*
2. **민감정보(병력) 국외이전 동의** — Claude API(미국) 전송. 현 `customer-agree`로 커버되나, 별도 동의서/마스킹 전송 필요? → AI 기능 전체 가부 결정
3. **컴플라이언스는 '심의 안전 배지' 금지** — 붙이는 순간 Foliio가 보증책임. '1차 보조, 최종책임 설계사' 면책 UI로 한정. 준법감시 자문 필요
4. **원탭 자동발송은 거짓 전제** — 카카오 인프라상 불가. MVP는 '클립보드 복사'로 정직하게. 알림톡 사전심의 경로는 별도
5. **바이럴 루프의 제3자 수집** — 지인 셀프진단 자동귀속 시 동의 없는 제3자 수집 소지. share_token 만료/회수/조회로그/robots 차단 수준 결정
6. **신입 콜드스타트** — 고객 5명·KPI 0인 신입이 첫 5분에 이탈. 부트스트랩(샘플 증권/상식 콘텐츠) 필요?

### 한눈 요약

> 분석은 끝점이 아니라 **미끼**다. 무료로 풀어 고객을 끌어오고, 그 위에 **AI가 작문·기억·발굴 노동을 대신**해 설계사가 영업 사이클을 무노동으로 돌리게 한다. 단, **법무(민감정보 국외이전 + 광고심의 면책)가 모든 AI 기능의 게이트**다 — 여기부터 푼다.
