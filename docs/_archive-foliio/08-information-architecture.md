# Foliio 영업지원 에디션 — 화면 구성 (정보구조 / IA)

> 어떤 페이지가 필요하고, 각 페이지에 무엇이 들어가는지만 한눈에. 작성 2026-06-18.

---

## 1. 전체 구조 한눈에

```
로그인 (카카오)
──────────────────────────────────
하단 메뉴 5개
  🏠 홈        오늘 할 일
  👥 고객      고객 목록·관리
  🎯 발굴      새 고객 만들기
  📊 분석      보장 분석·비교
  🙋 내정보    판촉물·설정·기록
──────────────────────────────────
전체 페이지
  홈 /home
  고객 /customer
    ├ 목록
    ├ 등록 (증권 사진 올리기)
    ├ 정보 동의
    └ 고객 상세
         ├ 보장 분석
         ├ 갈아타기 제안
         ├ 보장 공백
         └ 접촉 이력
  발굴 /prospect
    ├ 갈아타기 비교
    ├ 무료진단 링크 만들기
    └ 상품별 영업 가이드
  분석 /analysis
    ├ 담보 한눈표
    ├ 갈아타기 비교표
    └ 고객 메시지 만들기
  내정보 /me
    ├ 판촉물 만들기
    ├ 추천코드
    ├ 설정
    └ 동의·발송 기록

  [고객이 보는 화면] 링크로 열림
    ├ 진단 결과
    └ 비교안내서
```

## 2. 페이지별 — 무엇이 들어가나

### 로그인 `/login`
- 목적: 카카오로 로그인
- 화면에 있는 것
  - 서비스 한 줄 소개
  - [카카오로 시작하기] 버튼

---

### 🏠 홈 — 오늘 할 일 `/home`
- 목적: 켜자마자 "오늘 누구한테 연락할지" 바로 보이기
- 화면에 있는 것
  - 오늘 할 일 목록 (만기 임박 · 갱신 · 생일 · 동의 안 받은 고객)
  - 각 줄 = 고객 이름 + 할 일 + [연락] 버튼
  - 상단에 이번 달 남은 사용 횟수
- 주요 버튼: [고객 추가]

---

### 👥 고객 목록 `/customer`
- 목적: 내 고객 한눈에 보기 + 검색·정렬
- 화면에 있는 것
  - 검색창
  - 정렬 (만기순 · 최근순 · 이름순), 필터
  - 고객 카드 = 이름 · 나이 · 성별 · 월 보험료 · 만기 임박 표시 · 동의 상태
- 주요 버튼: [고객 추가]

### 👥 고객 등록 `/customer-create`
- 목적: 증권 사진/PDF 올리면 자동으로 채워주기
- 화면에 있는 것
  - 증권 올리기 (사진·PDF, 여러 장 가능)
  - 자동 인식 결과 → 빈칸만 채우는 입력폼 (이름 · 생년월일 · 성별 · 연락처)
  - 인식 못 한 담보는 직접 추가
- 주요 버튼: [올리기], [저장]

### 👥 정보 동의 `/customer-agree`
- 목적: 고객 정보(병력 등)를 AI로 분석하기 전에 동의 받기
- 화면에 있는 것
  - 어떤 정보를 왜 쓰는지 쉬운 설명
  - 동의 체크 + 확인
- 주요 버튼: [동의하기]
- 참고: 동의 전에는 분석 기능 잠김

### 👥 고객 상세 `/customer/:id`
- 목적: 한 고객의 모든 것 (탭으로 전환)
- 화면에 있는 것
  - [보장 분석] 탭 → 담보 한눈표
  - [갈아타기 제안] 탭 → 비교표
  - [보장 공백] 탭 → 부족한 담보 목록
  - [접촉 이력] 탭 → 언제 연락·발송·열람했는지
  - 상단에 동의 상태 표시
- 주요 버튼: [제안 만들기], [메시지 보내기]

---

### 🎯 발굴 — 갈아타기 비교 `/prospect/switch`
- 목적: 기존 보험 가진 사람에게 더 나은 제안 (안전하게)
- 화면에 있는 것
  - 기존 vs 제안 나란히 비교표 (담보 · 보험료 · 차액)
  - 손해보는 부분 자동 경고 (해지 환급금 손해 · 면책기간 다시 시작 등)
  - 고객 확인 체크칸
- 주요 버튼: [비교안내서 만들기], [PDF 저장]

### 🎯 발굴 — 무료진단 링크 `/prospect/self-check`
- 목적: 고객이 직접 입력하는 무료진단 링크 만들어 카톡으로 보내기
- 화면에 있는 것
  - 링크 만들기 (내 추천코드 자동 포함)
  - 만든 링크 목록 + 누가 들어왔는지 표시
- 주요 버튼: [링크 만들기], [카톡 공유]

### 🎯 발굴 — 상품별 영업 가이드 `/prospect/playbook`
- 목적: 상품(암 · 운전자 · 어린이 등)별로 누구한테 언제 어떻게 파는지
- 화면에 있는 것
  - 상품 목록
  - 상품별 = 누구를 · 언제 · 첫 멘트 · 거절 대응
- 주요 버튼: [멘트 복사]

---

### 📊 분석 — 담보 한눈표 `/analysis/:id`
- 목적: 100개+ 담보를 한 화면에, 있는 것/빈 것을 색으로
- 화면에 있는 것
  - 카테고리(사망 · 암 · 뇌 · 심장 · 실손 …) × 담보 격자표
  - 색 = 충분(파랑) / 부족(노랑) / 없음(회색)
  - 간략 보기 ↔ 상세 보기 전환
- 주요 버튼: [제안 만들기], [고객에게 공유]

### 📊 분석 — 갈아타기 비교표 `/analysis/:id/compare`
- 목적: 기존 vs 제안 비교 (발굴의 갈아타기 비교와 같은 화면)
- 화면에 있는 것
  - 담보별 기존/제안/차액
  - 손해 경고 + 고객 확인 칸
- 주요 버튼: [비교안내서 만들기]

### 📊 분석 — 고객 메시지 만들기 `/analysis/:id/notify`
- 목적: 분석 결과로 보낼 카톡 멘트 자동으로 뽑기
- 화면에 있는 것
  - 멘트 3종 (부드럽게 / 숫자 강조 / 급하게)
  - 직접 수정 가능 + 공유 링크 자동 첨부
- 주요 버튼: [복사], [카톡 열기]

---

### 🙋 내정보 `/me`
- 목적: 내 도구·설정 모음
- 화면에 있는 것
  - 판촉물 만들기 (명함 · 팜플렛 등 기존 템플릿)
  - 추천코드 (내 코드 · 들어온 사람)
  - 설정 (알림 · 구독 · 계정)
  - 동의·발송 기록 (언제 누구에게 동의받고 보냈는지)

---

### 📱 고객이 보는 화면 (링크로 열림)
> 설계사가 보낸 링크를 고객이 폰으로 여는 화면. 설계사용 메뉴/헤더 없이 깔끔하게.

- **진단 결과** — 본인 보장에서 부족한 곳을 쉬운 말로
- **비교안내서** — 기존 vs 제안 비교, 맨 아래 [무료진단 받기] / [상담 신청] 버튼

---

## 3. 모든 화면 공통 요소
- 상단: 이번 달 남은 사용 횟수 (무제한이면 ∞)
- 고객 정보 동의 상태 표시
- AI가 만든 글 아래엔 항상 작은 안내문 ("AI 초안 · 최종 확인은 설계사")
- 사용 횟수 다 쓰면 → 업그레이드 안내 팝업
- 알림 벨 (만기 · 동의 · 소개 등)

---

## 4. 먼저 만들 화면 (순서)
1. 로그인 → 홈
2. 고객 등록(증권 올리기) → 정보 동의
3. 담보 한눈표
4. 갈아타기 비교표
5. 고객 메시지 만들기
6. 고객 상세 (보장 공백 포함)

→ 그다음: 무료진단 링크 · 영업 가이드 · 추천코드 · 판촉물
