# 문서 05 — 상품군별 신규고객 발굴 플레이북

> **PM 핵심 지시 반영**: 보험설계사는 매달 실적(quota)을 채워야 하고, 그래서 **신규 고객 발굴이 영업의 진짜 병목**이다. 설득보다 *찾는 것*이 어렵고, 발굴 방식은 **상품군마다 완전히 다르다.** 그 전부를 지원한다.
>
> ⚠️ **중요 보정**: 이 문서는 8개 상품라인 전문가 에이전트 결과 + PM이 추가로 알려준 **한국 발굴의 현실(DB영업=갈아타기/승환 + 지인·소개)** 을 합쳐 다시 썼다. 단순 '인바운드 콘텐츠 미끼'가 아니라, 현장 주력인 **갈아타기 영업을 합법·전문적으로 무장**시키는 것이 1순위다.

---

## 1. 왜 발굴인가 — 그리고 한국 발굴의 진짜 모습

설계사는 위촉직(개인사업자)이라 **매달 새 계약을 못 만들면 소득이 0에 수렴**한다. 관리 고객은 연차마다 쌓이지만, 신규 발굴이 멈추면 끝이다. 그런데 현장에서 신규를 만드는 방식은 교과서의 '인바운드 마케팅'이 아니라 **세 가지 현실 모드**다:

| 발굴 모드 | 현장 비중 | 무엇인가 | Foliio의 역할 |
|---|---|---|---|
| **① DB영업 = 갈아타기(승환)** | **가장 큼** | 기존 가입자를 찾아 **기존 보험을 해약·재설계해 내 상품으로 재판매** | 기존 vs 제안 **비교분석 = 이미 보유한 무기**. 단 부당승환 컴플라이언스가 생명 → §2 |
| **② 지인·소개** | 큼 | 아는 사람 + 그 사람의 소개. 만족 고객이 단톡방으로 확산 | 진단링크 + **추천코드 자동귀속(referred_by)** 으로 소개를 데이터화 → §3-④ |
| **③ 발로 뛰기 / 인바운드 미끼** | 소수 정예만 | 방문·팜플렛·강연·SNS 콘텐츠로 신규 유입 | promotion 콘텐츠 엔진 + 심의 게이트 → §3-⑤. **열정적인 소수**만 함 |

> **결론**: 제품의 1순위는 ③(콘텐츠 바이럴)이 아니라 **①(갈아타기)을 안전하고 전문가답게 만드는 것**. 대부분의 설계사가 이미 ①을 하고 있고, Foliio의 분석/비교 엔진이 정확히 그 도구다. ②는 자동 귀속으로 복리화, ③은 소수 정예의 성장 옵션.

## 2. ⚠️ 갈아타기(승환)의 양날 — 부당승환 규제를 *기능*으로 뒤집기

### 문제: 현장 주력 영업 = 규제 1순위

기존 계약을 해약시키고 새 계약을 파는 것을 **승환계약**이라 하고, **보험업법 제97조가 부당승환을 명시적으로 금지**한다. 핵심 골자(정확한 기간·요건은 준법 자문 확정 필요):

- 신·구 계약의 **중요사항 비교안내(비교안내확인서)** 없이 갈아타게 하면 위법 소지.
- 기존계약 소멸 전후 **일정 기간 내** 유사계약 체결 시 **부당승환으로 추정** → 입증책임이 설계사에게.
- 고객에게 불리한 승환(보장 축소·면책 재기산·해지환급 손실)은 불완전판매·민원·환수의 핵심 원인.

즉, **설계사가 매일 하는 영업이 가장 위험한 영업**이다. 그리고 대부분은 이걸 구두로 대충 넘긴다.

### 반전: 법이 요구하는 걸 Foliio가 자동으로 만들어 준다 = 진입 무기

> **"당신이 매일 하는 갈아타기, 합법적으로·전문가답게. Foliio가 기존 vs 제안 비교안내서를 자동 생성해 당신을 보호합니다."**

규제(비교설명 의무)를 → **기능(자동 비교안내서 + 컴플라이언스 방패)** 으로 전환한다. 이게 범용 AI·캔바·엑셀이 **절대 못 주는** 차별점이다.

**자동 비교안내서 엔진** (MVP 핵심 기능 후보):
- 입력: 기존 증권 OCR + 제안 설계.
- 처리: 8케이스 분석엔진이 보장·보험료·해지환급·면책 변화를 항목별 대조 → Claude가 **고객이 불리해지는 지점을 명시**(보장 축소/면책 재기산/환급 손실) → 컴플라이언스 게이트가 부당승환 위험 신호를 경고.
- 출력: **법정 비교안내서 양식** + 고객 서명용 확인 동선 + 설계사 보관 로그.
- 효과: 합법 방패 + 고객 신뢰("이 분은 손해까지 짚어준다") + 민원·환수 방어.

⚠️ **레드라인**: Foliio는 '갈아타기를 부추기는' 도구가 아니라 **'정당한 승환만 안전하게 돕는'** 도구로 포지셔닝한다. 고객에게 불리한 승환은 경고로 **막는** 쪽이 장기 신뢰·법적 안전 모두에 맞다. "심의/합법 보증 배지"는 절대 붙이지 않음(보증책임 전가 → §6).

## 3. 공통 발굴 엔진 — 모든 상품라인이 함께 쓰는 인프라

라인별로 다른 건 *타겟·트리거·채널*이지, *엔진*이 아니다. 아래 5개는 8개 라인 전부가 공유한다(과잉 라인특화 방지).

### ① 갈아타기 안전 비교안내 엔진 — §2. *모든 라인의 1순위 무기*
기존 가입자 대상 영업의 표준 도구. 특히 **건강·암·실손(장기인보험)** 에서 압도적으로 강력(담보 비교가 복잡 = AI 가치 최대).

### ② 기존고객 보장공백 자동 스캐너 — *"다음 판매각도"를 AI가 찾아준다*
- 입력: 설계사의 Customer DB(기존 가입보험 `portfolio_type=1` + 연령·성별·직업위험·가족구성).
- 처리: 8케이스 분석엔진이 라인별 공백(사망보장 0 / 운전자보험 미보유 / 간병 슬롯 빔 / 일상배상 특약 없음)을 자동 플래그 + 가입연령 상한·갱신월 임박으로 우선순위 스코어링.
- 출력: **"이번 주 연락할 고객 TOP N"** + 각자 빈틈 한 줄 + 추천 라인. → 신규 발굴 비용 0의 최우선 풀.

### ③ 생애이벤트·트리거 캘린더 — *"지금 연락할 명분"*
발굴의 본질은 **타이밍**이다. 결혼·출산·내집마련·이직·자녀입학·생일(보험나이 인상)·만기갱신·연말정산·법규변화(민식이법 등)·검진결과를 자동 감지해 "오늘 접근할 고객 + 미끼 멘트"를 푸시. (재활용: `notifymembership` 크론 패턴 + SENS/카카오/인앱.)

### ④ 바이럴 진단링크 + 추천코드 귀속 — *지인·소개를 데이터로*
고객이 받은 `share_token` 진단링크 하단에 담당 설계사 추천코드 CTA → 지인이 셀프진단 → `referred_by` 자동 귀속. 만족 고객이 **단톡방에 붙여넣는 1메시지 패키지**를 AI가 생성. (한국 발굴 ②의 자동화.)

### ⑤ 미끼 콘텐츠 생성기 + 심의 게이트 — *소수 정예의 ③을 누구나*
promotion 14종 템플릿 + Claude로 라인·트리거별 정보형 카드/SNS/인포그래픽 자동 생성. **모든 외부 배포물은 보험광고 심의 게이트 통과분만 발행**(보장수치·상품명·수익률·단정표현 자동 필터).

## 4. 상품군별 발굴 플레이북 (8개 라인)

라인을 **발굴 성격**으로 묶었다. 각 라인의 *누구를·언제·어디서·미끼·핵심거절·AI지원·연계판매·컴플라이언스*.

### 묶음 A. 갈아타기(승환) 핵심군 — DB영업의 주전장

#### A-1. 건강·암·진단비 (장기인보험, 실손 포함) — *손해 설계사 주력 quota*
| 항목 | 내용 |
|---|---|
| 누구를 | 구실손(2009·2017 이전) 보험료 폭탄 40~60대 / 첫·둘째 출산 가정 / 단체실손 의존 직장인 / 가족력 암 30~50대 / 검진 경계수치 직후 표준체 막차 |
| 구매 트리거 | 구실손 갱신 인상통지, 검진결과(3~4·11~12월), 가족 암진단, 보험나이 인상 직전, 단체실손 종료(이직/퇴직), 4세대 전환 프로모션, 연말정산 |
| 발굴 채널 | **기존 고객 DB(승환 1순위)**, 지역 맘카페·육아카페, 검진센터·내과 인근, 가족 소개망(실손=가족단위), share_token 단톡방 바이럴 |
| 진입 미끼 | **"지금 실손 4세대 전환하면 얼마 아껴지는지 1분 무료진단"** → 본인이 입력→자동 손익분기 리포트. 거절감 0 |
| 핵심 거절 | "구실손이 좋다"(갱신폭탄 미인지), "단체로 충분", "갈아타면 손해 아니냐"(손익분기 계산 필요) |
| **Foliio AI** | **구실손 vs 4세대 전환 손익 자동진단**(§2 비교안내서) + 기존고객 갱신월/검진시즌 발굴 큐 + 가족 확장 소개 매핑 |
| 연계판매 | 관문상품 → 가족력시 고액암진단비 상향 → 가장 종신/정기 → 50대+ 간병/요양 |
| ⚠️ 컴플라이언스 | 갈아타기 = **부당승환 직격**(§2 필수). 무작위 발굴 금지→자발적 진단입력 구조. 증권=민감정보 국외이전 동의 |
| 승환 적합도 | ★★★★★ (갈아타기의 본진) |

#### A-2. 생명보험 (종신·정기·CI·변액) — *사망보장 공백 = 신규 발굴 핵심*
| 항목 | 내용 |
|---|---|
| 누구를 | 35~48세 주담대 1억+ & 유자녀 가장 / 신규주택 구입자 / 40~55세 자영업·법인대표(상속·키맨) / 자산가 10억+(상속세 재원) / **"손보 실손·암만 있고 사망보험금 0~3천"인 기존고객** |
| 구매 트리거 | 결혼·첫출산(최강), 주담대 실행, 지인 사망/중대질병, 법인설립·매출성장, 상속 인지, 연말정산 |
| 발굴 채널 | **기존 DB 내 사망보장 공백자(발굴비용 0)**, 산모교실·학부모모임, 부동산·법무사·대출모집인 제휴, 세무사 제휴(법인·상속), 소개영업 |
| 진입 미끼 | **"지금 잘못되면 남은 가족이 빚 갚고 생활할 돈, 얼마나 준비됐는지 1분 진단"** → 필요보장액 vs 현재 사망금 갭 그래프. 법인=대표유고 채무, 자산가=상속세 현금재원 변형 |
| 핵심 거절 | "내 죽고 받는 돈 왜 지금"(→정기·'가족'프레임), "실손 있는데 또"(사망≠의료 갭 분리), "종신 비싸다"(정기+핵심종신) |
| **Foliio AI** | 사망보장 공백 자동 스캐닝(현재 사망금 집계 vs 권장 필요보장액) + 생애사건 트리거 알림 + 상속·법인 시나리오 1페이지 브리프(세무 단정 회피 디스클레이머) |
| 연계판매 | 실손만 보유 고객 → 종신/정기 업셀(가장 자연) → 배우자 정기 → 자녀 어린이 → 법인은 화재·배상 연계 |
| ⚠️ 컴플라이언스 | **GA 고수당(종신) 편향 주의** — 고객이익 우선 위반 리스크. 상속·세무 단정 금지. 광고심의·국외이전 |
| 승환 적합도 | ★★★☆☆ (주로 공백채우기 신규, 일부 갈아타기) |

### 묶음 B. 미끼·의무가입군 — 저단가 진입 + 끼워팔기

#### B-1. 운전자보험 — *저단가·고빈도 진입 미끼*
| 항목 | 내용 |
|---|---|
| 누구를 | 자동차보험만 있고 운전자보험 미가입자(압도적 다수), 영업·배달·외근 고빈도 운전자, 스쿨존 학부모, 면허취득·신차 출고자 |
| 구매 트리거 | **자동차보험 갱신월 D-30(최강)**, 면허취득·신차구매, 민식이법·음주단속 뉴스, 명절 장거리·신학기, 사고 직간접 경험 |
| 발굴 채널 | **기존 DB 중 자동차보험O·운전자보험X 자동리스트(가장 싸고 빠름)**, 신차 동선 제휴(카센터·썬팅·중고차), 배달 라이더 단톡방, SNS '자동차 vs 운전자 차이' 콘텐츠 |
| 진입 미끼 | **"자동차보험은 상대 피해만, 내 형사합의금·변호사비는 0. 스쿨존 한 번이면 수천만원. 구멍 있는지 1분 무료점검"** |
| 핵심 거절 | "자동차보험으로 안 되나"(대물/대인 vs 형사 구분), "운전 조심해서"(중과실 무관), "인터넷이 싸다"(저단가라 더 심함) |

- **신입 트랙** = ① 비교안내서(갈아타기 무기) + ② 공백 스캐너 + 진입미끼 라인(운전자·배상·펫) + 콜드스타트(샘플 증권·상품지식·거절처리 스크립트).
- **중견 트랙** = ② 공백 스캐너(쌓인 DB에서 생명·간병 업셀) + 만기 워치독 + 고객 A/S·사후관리 + 고아계약 승계.

## 7. 리스크 & 컴플라이언스 종합

발굴은 **개인정보·모집질서·광고심의**가 동시에 걸리는 최고 난이도 구간이다.

| 리스크 | 내용 | 대응 |
|---|---|---|
| **부당승환** | 갈아타기 영업이 보험업법 제97조 직격. 비교안내 없이 승환 시 위법·추정·환수 | **자동 비교안내서**(§2) + 부당승환 위험신호 경고 + 불리한 승환 차단. '합법 배지' 금지 |
| **DB명단 출처** | 명단 구매·무단수집·타사 DB 재활용 = 개인정보보호법 위반 | **본인이 자발적 입력(self-input) + 동의 선통과** 구조만. 기존고객은 가입시 마케팅 동의 범위 내 |
| **무작위 발굴** | 콜드콜·스크래핑 = 동의 없는 수집 | 동의기반 **인바운드(진단링크 클릭)** 로만 전환 |
| **무자격 모집(제휴)** | 병원·펫샵·부동산 직원 권유 = 무등록 모집 / 수수료=리베이트 | 제휴처는 **QR·정보비치까지만**, 청약은 등록 설계사. 대가는 광고·장소제공 구조 |
| **광고심의** | AI 생성 외부 콘텐츠가 보장·수익률·비교·단정 표현 시 사전심의 대상 | **심의 게이트 통과분만 발행**, 금칙어·단정 자동필터, 심의필·소속 표기 |
| **민감정보 국외이전** | 병력·소득을 Claude(Anthropic, 美) 전송 | 진단/OCR 진입 시 **국외이전 동의 게이트**. 동의 전 AI 호출 차단 |
| **아동정보** | 만 14세 미만 = 법정대리인 동의 | 어린이 진단에 부모 동의 동선 강제 |
| **GA 고수당 편향** | 종신 위주 권유 = 고객이익 우선 위반 | 추천 로직이 수당 아닌 **공백/적합도** 기준임을 명시 |

### PM 확인 대기 (오픈 퀘스천)
1. **승환 비교안내·법적 처리를 현장에서 지금 어떻게 해?** 회사가 양식을 주나, 구두로 넘기나? → 자동 비교안내서의 자동화 깊이 결정.
2. **DB(명단)은 어디서 와?** 회사 제공 / TM / 구매 / 인맥? → 합법 경계 + 제품이 합법 DB만 받는 구조 설계.
3. **결제 주체** = 개인 설계사 월 구독 vs 지점/팀 번들? → 가격·온보딩.
4. 부당승환 정확 요건(기간·확인서 양식) → **준법감시·보험업법 전문가 자문 선결**(자동 비교안내서 출시 게이트).

## 8. 한눈 요약

> 한국 발굴의 현실은 **DB영업 = 갈아타기(승환)** 다. Foliio의 기존vs제안 비교 엔진이 **이미 그 무기**고, 부당승환이라는 법적 지뢰를 **자동 비교안내서**로 뒤집으면 누구도 못 따라오는 차별점이 된다. 라인마다 *타겟·트리거·채널·미끼*는 다르지만(§4), *엔진은 공통*(§3)이다. 신입은 진입미끼+갈아타기로 quota를 채우고, 중견은 쌓인 DB 공백 스캐너로 객단가를 올린다. **발굴 기능을 MVP로 끌어올리되**(§6), 모든 발굴은 **동의기반 인바운드 + 심의 게이트 + 부당승환 방패**를 통과해야 출시된다(§7).
